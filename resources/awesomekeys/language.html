Language

This plugin can handle mutliple languages. But it relies heavily on the community, so if you don't see your language, please submit your own translation file following the steps described below. All languages files can be found here.

How to create a new language file

First, copy the default language file lang_en.json, then name it with the following convention : "lang_" + + ".json".

Language code is two or three letters representing the language (de, es, it, nl, ru, cn, jp, ar...). Please keep in mind that there is no warranty that your language file will be correct ; if not, then please submit a correction. English and French are the only officialy maintained languages for now.

How to make a language file "recommended"

Your language file must have received a good feedback on github. Then your file will be moved from the "unverified" to the "recommended" folder.
