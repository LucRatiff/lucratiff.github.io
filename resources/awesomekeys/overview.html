AwesomeKeys

AwesomeKeys is a Bukkit plugin, which allows you to lock doors and containers, and open it with your own keys, that you can give to your friends, or with a 9 digit code. When a block is locked, the only way to access it is to have the propper key or the right code.

You can also add descriptions to your keys to find them easily, and add greeting messages when a door is unlocked. Cherry on the top, you have nice particles for each action.

Customizable

AwesomeKeys is deeply customizable through the configuration file : you can decide which block can be locked or can be a key, wether protection plugins have the priority over accessing a locked block or not, having particles or not... All you have to know about configuration is explained in details in the documentation. This plugin is also mutli-languages : see the Language section.

In a near future, an API will be provided to developpers.

Which blocks can be locked ?

All doors, fence gates, shulkers, chests and all other containers (except hoppers and entities). You can customize the list in the configuration file.

Which item can be a key ?

Any item. By default, the key is the wooden stick. When the item is affected as a key, there are special tags applied to it. You can define mutliple items and durabilities as keys. You may prefer an item that players cannot obtain. And maybe an item on wich you can add a new texture : please refer to the Keys section.

How to enter a code ?

If the block is locked with a code, it will display an 3x3 inventory on access. Just click on the numbered wool blocks.
